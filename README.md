## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/Snokei/OneInstall.git
cd OneInstall
```

### Install dependencies

```bash
npm install
```

### Start development

```bash
npm run electron:dev
```

### Build production

```bash
npm run dist
```

The generated installer will be available in:

```
release/
```
